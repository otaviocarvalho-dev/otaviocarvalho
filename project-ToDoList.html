To Do List

O que é o projeto To Do List?

O projeto To Do List nada mais é do que uma lista de tarefas web. Serve para o usuário definir uma sequência de tarefas a ser feita, e quando concluída, tem a opção de marcar ou excluir a tarefa.

Ferramentas utilizadas na construção do projeto:

  • JavaScript
  • HTML
  • CSS
